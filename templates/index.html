Clear Maps Tutorial

Navigation Bar

The navigation bar is found at the top of any page on the Clear Maps site and is used to go to a different page on the site. You click the page you would like to go to and it will take you there.

Accessibility Page

My site contains accessibility settings which may be helpful to enable to aid use of the site. If you would like to check them out, please click the button below. Once you have applied the desired settings, please return to this page using the navigation bar as described above.

Click to go to Accessibility Page

Login and Registration Pages

Registration

The registration page is shown below. To view the mapping and journey pages, you must register an account. The page has 4 boxes to enter a username, password, email and name. The page will provide you with error messages and guide you through the process if something goes wrong.

Login

Once you have registered, you can login on this page. Please enter the same credentials as when you registered. There is also a "forgot password" option if you need to reset your password.

Forgot Password

If you can't remember your password, please click "forgot password" on the login page. You can then enter your email on the forgot password page and a new password will be emailed to you. This password replaces your original password so you can use it to login with your old username. Once logged in, please go to the profile page and change the temporary password issued to a new memorable one.

Mapping Page

Mapping

This page is used to interact with the map, create routes between points and save journeys for easy access later.

  • To get started, select the textboxes and enter:
    • Start Address and Postcode
    • End Address and Postcode
  • Then select the "Get Route" button to display the route on the map. A message will confirm whether this action has succeeded.

The map below will update to display the first location you entered and a marker will be displayed on this point. There will be a blue line leading away from the point, towards the destination which you would follow for the route to the destination.

After creating a few journeys, an autofill feature will show some previously used addresses below the textbox you are typing in which you can select if you are going to that address to speed up location selection.

Save Journey

To open this feature, please select the "Toggle Save Journey Options" button. This will display a few additional textboxes. To close this section, press the button again.

  • To add a new saved journey started, select the textboxes and enter:
    • A memorable name for each addresses entered above
    • Click on the checkbox if you would like to favourite this address (for use in the autofill feature)
    • A memorable name for the journey between the two addresses. e.g. "Park Walk" or "Walk to town"

Then select the "Save Journey" button. If there are any errors, they will be displayed, otherwise, the section will close and the journey has been successfully saved.

Journey Management Page

This page can be reached by clicking on the "Journey" option on the navigation bar. It has a table which displays every journey you have saved on the map page. You have management options beside the journey such as "Load Journey" and "Delete"

Load Journey

When this button is pressed, the journey listed in the table beside the button will be loaded onto the map. After pressing, you will be moved to the map page where you can view the journey.

Delete

When this button is pressed, the journey listed in the table beside the button will be deleted. After pressing, you will recieve a message asking you to confirm the deletion. If "OK" is selected, the journey will be deleted. If "Cancel" is selected, the journey will not be deleted.

Profile Page

This page can be reached by clicking on the "Profile" option on the navigation bar.

Change Username

This feature is used to update the username on your account. You can enter a new desired username in the text box and press "Change". A success / error message will be displayed based on the outcome of the change.

Change Password

This feature is used to update the password on your account. You must enter your old and new password in the text boxes and press "Change". The site will display a success / error message.

Log Out Button

This button can be found on the far right of the navigation bar when logged in. It will log you out of the site and make the map, journeys and profile page inaccessible until you login again.
